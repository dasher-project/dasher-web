Integrated Demonstration

The user interface can also be added as a a custom keyboard:

  1. Open the device Settings
  2. Navigate through General, Keyboard, Keyboards, to Add New Keyboard.
  3. Add the keyboard from this application.

The user interface can then be used as the keyboard in any application on the device.

This is a proof of concept application and isn't intended for production use.
